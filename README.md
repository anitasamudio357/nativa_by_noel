# NATIVA PÁG

Proyecto web para NATIVA PÁG

## Descripción

Este proyecto contiene las páginas web principales del sitio.

## Archivos

- `index.html` - Página principal
- `producto.html` - Página de producto
- `styles.css` - Estilos CSS
- `src/` - Carpeta con archivos adicionales

## Cómo usar

Abre `index.html` en tu navegador para ver la página principal.
