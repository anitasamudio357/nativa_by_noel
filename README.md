# NATIVA by Noel | Alimentación Consciente

Proyecto web para el lanzamiento de la línea **NATIVA by Noel**, una gama de productos a base de plantas que integra salud, bienestar animal e impacto ambiental bajo el lema: *"Pequeñas decisiones, gran impacto"*.

## Demo en vivo

🚀 **[Ver el proyecto en Vercel](https://nativa-by-noel.vercel.app/)**

## Descripción

El sitio web sirve como un escaparate digital para explicar el origen, la filosofía y el propósito de NATIVA, así como para detallar la información de los productos, su empaque sustentable (que se transforma en una experiencia de juego interactiva), y presentar a los diferentes personajes enfocados en la educación de la conciencia ambiental.

## Estructura de Archivos

- `index.html` - Página principal (landing page) que incluye las secciones de Inicio, Sobre Nosotros, Productos, Empaque, Personajes y El Juego Nativa.
- `styles.css` - Hoja de estilos para toda la aplicación.
- `main.js` - Scripts y lógica de interactividad (menús, animaciones, etc.).
- `src/` - Carpeta de multimedia / recursos adicionales (imágenes, gráficos vectoriales, visuales del juego).
- **Páginas de Productos:**
  - `ChocolateOscuro.html`
  - `FresaNatural.html`
  - `KiwiFresco.html`
  - `ArandanosSilvestres.html`

## Cómo ejecutar el proyecto

Este proyecto es estático, por lo que no requiere instalación de dependencias ni un servidor complejo para previsualizarse en modo básico.

1. Simplemente haz doble clic sobre o abre `index.html` en tu navegador web de preferencia para cargar la página de inicio.
2. Puedes navegar a través del menú de la cabecera para desplazarte a las diferentes secciones.
3. Al hacer clic en cualquiera de los productos desde la sección principal, serás dirigido al archivo HTML de ese sabor en particular.
